# GitHub Staging Directory

This directory contains files that are safe to push to GitHub.
All local development files remain in the parent directory.

## Structure:
- templates/metal-node/ - Metal validator template
- README.md - Project overview
- CHANGELOG.md - Project changelog  
- PROGRESS.md - Project progress
- .gitignore - Git ignore rules

## Usage:
- Work in the main directory for local development
- Copy files to staging/github/ when ready for GitHub
- Never work directly in staging/github/ to avoid losing local files
